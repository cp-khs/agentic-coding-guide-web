# Claude Code를 활용한 에이전틱 코딩 실습 튜토리얼

> **사전 준비**: 이 튜토리얼을 시작하기 전에 [에이전틱 코딩 가이드](/)와 [Claude Code 소개](/claude-code)를 먼저 읽어주세요.
> **목표**: Claude Code로 TaskFlow 프로젝트를 구성부터 구현까지 완수하기
> **예상 소요 시간**: 약 4시간

---

## 1.1 프로젝트 개요

**TaskFlow**는 팀 단위 태스크 관리 웹 애플리케이션이다. 칸반 보드 스타일의 UI로 태스크를 관리하며, 팀원 배정, 상태 추적, 댓글 기능을 제공한다.

이 프로젝트는 에이전틱 코딩을 학습하기에 적합한 규모와 복잡도를 갖추고 있다. 레거시 방식으로는 백엔드·프론트엔드·DevOps·QA 각 1명이 필요한 규모지만, Claude Code를 활용하면 1명의 개발자가 약 4시간 내에 완성할 수 있다.

## 1.2 기술 스택

| 영역 | 기술 |
|------|------|
| 백엔드 | Python 3.12, FastAPI, SQLAlchemy 2.0, Alembic |
| 프론트엔드 | Next.js 15 (App Router), TypeScript, Tailwind CSS |
| 데이터베이스 | PostgreSQL 16 |
| 인증 | JWT (python-jose, passlib) |
| 인프라 | Docker, Docker Compose |
| 테스트 | pytest (백엔드), Vitest (프론트엔드) |
