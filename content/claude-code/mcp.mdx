# MCP 서버 연동

> **MCP(Model Context Protocol)**는 Claude Code가 외부 도구 및 서비스와 통신하기 위한 표준 프로토콜이다. Anthropic이 설계한 오픈 표준으로, Claude가 브라우저, 데이터베이스, 이슈 트래커 등 다양한 외부 시스템과 상호작용할 수 있게 해준다.

---

## MCP란?

기본 Claude Code는 파일 읽기/쓰기, 쉘 명령 실행, 웹 검색 같은 내장 도구만 사용할 수 있다. MCP를 통해 **외부 서비스와의 연동을 플러그인 방식으로 확장**할 수 있다.

MCP 서버를 추가하면 Claude Code에서 다음을 직접 수행할 수 있다.

- 브라우저를 자동 조작하거나 스크린샷을 찍는다
- GitHub, Linear, Jira 이슈를 직접 생성·수정한다
- 데이터베이스에 쿼리를 실행한다
- 최신 라이브러리 문서를 실시간으로 검색한다

## MCP 서버 추가

### CLI로 추가

```bash
# 사용자 전체에 등록 (모든 프로젝트에서 사용 가능)
claude mcp add --scope user <이름> -- <실행 명령>

# 현재 프로젝트에만 등록
claude mcp add --scope project <이름> -- <실행 명령>

# 예시: Playwright MCP 서버 추가
claude mcp add --scope user playwright -- npx @playwright/mcp@latest

# 예시: Context7 MCP 서버 추가
claude mcp add --scope user context7 -- npx @upstash/context7-mcp

# 등록된 MCP 서버 목록 조회
claude mcp list

# 특정 MCP 서버 상세 정보
claude mcp get playwright

# MCP 서버 제거
claude mcp remove playwright
```

### settings.json으로 추가

`~/.claude/settings.json` 또는 `.claude/settings.json`에 직접 설정할 수 있다.

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/mcp@latest"]
    },
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp"]
    }
  }
}
```

## stdio vs HTTP 방식

MCP 서버는 두 가지 통신 방식을 지원한다.

| 구분 | stdio (로컬 프로세스) | HTTP (원격 서버) |
|------|----------------------|-----------------|
| 실행 방식 | Claude Code가 직접 프로세스를 시작 | 별도로 실행 중인 서버에 HTTP 연결 |
| 설정 | `command`, `args` 지정 | `url` 지정 |
| 용도 | 로컬 도구, CLI 기반 서비스 | 원격 API, 팀 공유 서버 |
| 보안 | 로컬 실행이므로 상대적으로 안전 | 네트워크 보안 설정 필요 |

**HTTP 방식 설정 예시**:

```json
{
  "mcpServers": {
    "remote-tool": {
      "url": "http://localhost:3000/mcp",
      "headers": {
        "Authorization": "Bearer my-token"
      }
    }
  }
}
```

## 주요 MCP 서버

### 개발 도구

| MCP 서버 | 패키지 | 기능 |
|----------|--------|------|
| Playwright | `@playwright/mcp` | 브라우저 자동화, E2E 테스트, 스크린샷 |
| Chrome DevTools | `@anthropic-ai/mcp-chrome-devtools` | 크롬 개발자 도구 연동 |
| Context7 | `@upstash/context7-mcp` | 최신 라이브러리 공식 문서 검색 |
| Sequential Thinking | `@modelcontextprotocol/server-sequential-thinking` | 단계적 추론 및 복잡한 분석 |

### 협업 도구

| MCP 서버 | 패키지 | 기능 |
|----------|--------|------|
| GitHub | `@modelcontextprotocol/server-github` | 이슈, PR, 코드 검색 |
| Linear | `@linear/mcp-server` | 이슈 트래커 연동 |
| Jira/Confluence | Atlassian MCP | Atlassian 제품군 연동 |
| Slack | `@modelcontextprotocol/server-slack` | 메시지 읽기/전송 |

### 데이터

| MCP 서버 | 패키지 | 기능 |
|----------|--------|------|
| PostgreSQL | `@modelcontextprotocol/server-postgres` | DB 쿼리 실행 |
| SQLite | `@modelcontextprotocol/server-sqlite` | SQLite DB 연동 |
| Filesystem | `@modelcontextprotocol/server-filesystem` | 파일 시스템 확장 접근 |

## 환경 변수 설정

MCP 서버에 API 키나 비밀값을 전달할 때는 `env` 필드를 사용한다.

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_token"
      }
    }
  }
}
```

> **주의**: API 키를 settings.json에 직접 기입하는 경우, 해당 파일을 `.gitignore`에 추가하거나 `.claude/settings.local.json`(자동 gitignore됨)을 사용한다.
